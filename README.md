# Kali-Linux-WSL
This will tell you how to downlaod kali linux on the power shell of windows WSL version.


Open Powershell as an administrator and run:
wsl --install
 This installs ubuntu as a default

 Do
 wsl --status
To check whether or not it is installed

To set it as default:
wsl --set-default-version 2

To install Kali Linux run:
wsl --install -d kali-linux

Finally, to get it type in 
kali
